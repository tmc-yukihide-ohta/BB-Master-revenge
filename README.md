BB-Master (売買マスター)
レーヴ日進での不用品の円滑な売買・譲渡が行える

実際の挙動は以下のページで確認できる。
参考URL：https://bb-master-revenge-front.onrender.com/

インストール
フロントエンド、バックエンド共に各フォルダへ移動し下記コマンドを実行。

$ git clone https://github.com/your-username/your-project.git
$ cd your-project
$ npm install

使い方
フロントエンド、バックエンド共にサーバーを起動。
フロントエンド：　
```bash
$ npm start //書き換える

バックエンド：　
```bash
$ npm start //書き換える

機能
１：フロントエンドのトップページへ移動。
２：欲しい物があったら、コメントを記入。
３：譲渡依頼ボタンを押して完了。
